# Customer Retention & Churn Analysis Dashboard

## 📌 Project Overview
This project was completed as part of my Data Science & Analytics Internship. The objective was to analyze customer churn patterns and identify the key factors affecting customer retention using Power BI.

The dashboard provides actionable insights that can help businesses improve customer retention strategies and reduce churn.


## 🛠️ Tools & Technologies
- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)


## 📊 Dataset
*Dataset Used:* Telco Customer Churn Dataset

The dataset contains customer demographic information, subscription details, payment methods, contract types, tenure, monthly charges, and churn status.


## 📈 Dashboard Features
- Total Customers KPI
- Churned Customers KPI
- Churn Rate (%)
- Churn by Contract Type
- Churn by Internet Service
- Churn by Payment Method
- Customer Tenure vs Churn
- Interactive Slicers for:
  - Contract Type
  - Internet Service
  - Payment Method


## 🔍 Key Insights
- Overall customer churn rate is *26.58%*.
- Customers with *Month-to-Month* contracts exhibit the highest churn.
- Customers using *Electronic Check* have the highest churn rate.
- *Fiber Optic* customers are more likely to churn than DSL users.
- Most customers who churn leave during the early months of their subscription.


## 💡 Business Recommendations
- Encourage customers to switch from Month-to-Month to long-term contracts.
- Improve customer onboarding and engagement during the first year.
- Investigate the reasons behind high churn among Fiber Optic customers.
- Promote automatic payment methods to improve customer retention.
