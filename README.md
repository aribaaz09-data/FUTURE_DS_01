#Superstore Sales Analysis

Project Overview:
This project analyzes sales data from a retail superstore using Power BI. The goal is to identify salestrends, profit performance, customer segments, and regional insights through interactive dashboards.

Tools Used:

*PowerBi

*Microsoft Excel

Dataset:
The dataset contains information about-
Orders, Sales, Profit, Customers, Products, Regions

Key Insights:

-Sales performance across different regions

-Profitability analysis

-Top-performing product categories

-Customer segment analysis

-Monthly sales trends
